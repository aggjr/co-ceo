# Telas obsoletas

Ficheiros HTML autónomos antigos que **não representam** o fluxo principal do produto. Mantêm-se para referência, comparação visual ou abertura em `file://` / servidor estático legado.

- **`apollo_grid.html`** — grelha “APOLLO COMMAND CENTER” (catálogo estático). O equivalente suportado no CO-CEO é **STOCKSPIN → Mix de Produtos** (`catalogGridView` no template).

Ao mover novas telas para aqui, ajuste caminhos relativos para `../data/`, `../curtain_production_data.js`, etc., e deixe na raiz do repositório um redirecionamento ou nota se ainda existirem links externos.
